CSS

CSSis stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files.

You should always include the lang attribute inside the html tag, to declare the language of the Web page. This is meant to assist search engines and browsers.
